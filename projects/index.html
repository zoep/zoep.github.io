  • Compiler for Llama

    During a compilers course at NTUA, along with 2 other students, I developed an optimizing compiler for Llama, a functional OCaml-like language. Llama features include type inference, higher-order functions, algebraic data types, patter matching and mutable references. The compiler performs various optimizations, such as copy propagation, common subexpression elimination, dead code elimination, tail call elimination and peephole optimizations.` (Code)

  • Applications of Logic in CS Project

    As a project for the course Applications of logic in Computer Science at NTUA, I implemented in Haskell, along with an other student, a toy interpreter for a lambda calculus variant that additionally performs Hindley–Milner type inference. (Code)

  • Advanced Topics in Database Systems Project

    A survey about security and cryptography in database systems. (Report)

  • Cryptography Project

    A library implementing basic operations on elliptic curves over prime fields, as well as the Elliptic Curve digital signature scheme and the Diffie-Hellman key exchange algorithm. Developed in OCaml in a team of 2 students. (Code)
